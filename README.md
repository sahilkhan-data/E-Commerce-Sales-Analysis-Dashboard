
# 📊 E-Commerce Sales Analysis Dashboard (Excel)

## 📌 Objective

E-Commerce Store wanted to create an annual sales report for 2025, so they could understand their customers better and grow more sales in 2026. This dashboard consolidates ~31,000 order records into a single interactive view for quick, data-driven decisions.

---

## ❓ Sample Business Questions Answered

- Compare sales and orders using a single chart
- Which month got the highest sales and orders?
- Who purchased more — men or women — in 2022?
- What are the different order statuses in 2025?
- List the top 5 states contributing to sales
- What's the relation between age and gender based on number of orders?
- Which channel is contributing to maximum sales?
- What is the highest-selling category?

---

## 🚀 Features

- 📈 Combo chart (bar + line) comparing **monthly Sales vs Orders**
- 🥧 Gender-wise sales split (**Men vs Women**)
- 👥 Age group vs Gender breakdown (Adult / Old / Teenager)
- 🗺️ **Top 5 states** contributing to sales
- 🛒 Sales channel-wise contribution (Amazon, Flipkart, Myntra, Meesho, Ajio, Nalli, Others)
- 📦 Order status breakdown (Delivered, Cancelled, Refunded, Returned)
- 🔄 Interactive **Slicers** for Month, Channel, and Category filtering
- 🎨 Clean single-page dashboard layout for quick insights

## 📊 Dataset

- **Records:** ~31,000 orders (2025)
- **Fields:** Order ID, Customer ID, Gender, Age, Age Group, Date, Month, Status, Channel, SKU, Category, Size, Qty, Currency, Amount, Ship City, Ship State, Ship Postal Code, Ship Country, B2B

---

## 💡 Key Insights

- 👩 **Women are more likely to buy** compared to men (~65%)
- 🗺️ **Maharashtra, Karnataka and Uttar Pradesh** are the top 3 contributing states (~35%)
- 🎂 **Adult age group (30–49 yrs)** contributes the most to orders (~50%)
- 🛒 **Amazon, Flipkart and Myntra** are the top-contributing sales channels (~80%)

### 🎯 Final Recommendation to Improve E-Commerce Store Sales
> Target **women customers aged 30–49 yrs** living in **Maharashtra, Karnataka and Uttar Pradesh** by promoting ads/offers/coupons on **Amazon, Flipkart and Myntra**.

## 📥 How to Use

1. Clone or download this repository
2. Open `E-Commerce Dashboard Excel.xlsx` in Microsoft Excel (2016 or later recommended)
3. Go to the **Dashboard** sheet
4. Use the **Month / Channel / Category** slicers to filter and explore the data interactively
